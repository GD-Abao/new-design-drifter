## 時隔六年的續作終於來了

2019 年的《噬血代碼》（Code Vein）以「動漫風格的黑暗靈魂」打出名號，雖然評價褒貶不一，但獨特的美術風格和角色創建系統讓它獲得一批忠實粉絲。

六年後，**《噬血代碼 2》** 於 2026 年 1 月 30 日正式發售。這次，Bandai Namco 帶來了開放世界、時空穿越劇情、全新搭檔系統 — 野心十足，但也帶來了一些問題。

## 劇情：時空穿越拯救世界

### 故事設定

Code Vein 2 的劇情與前作 **大部分獨立**，新玩家可以直接遊玩。

**背景設定：**

- 未來世界，人類與吸血鬼（Revenants）在瀕臨崩潰的地球上掙扎
- 新威脅「Luna Rapacis」出現，將 Revenants 變成恐怖的「Horrors」
- 玩家扮演「吸血鬼獵人」（Revenant Hunter）

**核心劇情：**

- 與時空旅行者「Lou」一起穿越回過去
- 目標是改寫命運，阻止災難發生
- 在兩個時代之間來回探索

### 劇情評價

**優點：**

- 獨立故事，不需要玩過前作
- 時空穿越設定有新意
- 角色塑造有深度

**缺點：**

- 部分劇情節奏拖沓
- 某些轉折可預測

## 遊戲玩法：開放世界的類魂體驗

### 最大變革：開放世界

這是系列最大的改變 — **完全開放世界**。

與前作和《噬神者》系列的關卡制不同，Code Vein 2 讓你：

- 從遊戲開始就能探索大部分地圖
- 三個主要目標可以任意順序挑戰
- 用摩托車快速移動（可以做出《阿乞拉》的經典甩尾）

### Jail 系統（血獄）

取代前作的「Blood Veil」，新的「Jail」系統是你的主要裝備：

**功能：**

- 啟用「Formae」— 超自然武器技能
- 執行「Drain Attack」— 伸出獸爪吸取敵人的「Ichor」
- Ichor 用於驅動 Formae 技能

**Formae 種類：**

- 強力武器攻擊
- 防禦護盾和增益效果
- 遠程投射攻擊
- 其他多種能力

### 搭檔系統（Partner System）

類似《艾爾登法環》的骨灰，但更進一步：

**兩種模式：**

| 模式     | 說明                       |
| -------- | -------------------------- |
| 召喚模式 | 搭檔獨立戰鬥，吸引敵人注意 |
| 同化模式 | 與搭檔融合，提升數值和火力 |

**實用性：**

- 隨時可用，不需要消耗品
- 可以讓搭檔吸引 Boss 注意力
- 同化模式適合輸出階段

### 武器系統

遊戲提供多種武器類型：

- 雙劍
- 大錘
- 刺刀
- 其他多種選擇

每種武器有不同的 Drain Attack 動作和 Formae 相性。

## 角色創建：系列招牌

### 更強大的捏臉系統

Code Vein 系列的角色創建一直是賣點，這次更進一步：

**新增功能：**

- 更細緻的體型調整
- 髮型可調整捲曲度、瀏海角度
- 更多服裝和配件選項

**玩家評價：**

> 「你可以輕鬆在角色創建花掉一小時，還沒看到任何遊戲內容。」

### 外觀變更

遊戲中可以隨時變更角色外觀，包括服裝和配件。

## 畫面與效能：最大問題

### 視覺風格爭議

Code Vein 2 從 Unreal Engine 4 升級到 **Unreal Engine 5**，但視覺風格引發爭議：

**批評聲音：**

> 「Code Vein 2 的視覺是前作的退步。美術風格完全不同，而且不是好的方向。前作有獨特的哥德動漫風格，續作的美感像是在轉換過程中迷失了。它看起來像任何其他通用的 UE5 動漫遊戲。」

**對比：**

| 項目   | 前作     | 續作         |
| ------ | -------- | ------------ |
| 引擎   | UE4      | UE5          |
| 風格   | 哥德動漫 | 較通用的動漫 |
| 獨特性 | 高       | 較低         |

### 效能問題

**首發災難：**

許多玩家報告嚴重的效能問題：

- 明顯的掉幀
- 卡頓和停頓
- 持續數小時的不穩定

這讓許多搶先體驗的玩家留下負面第一印象。

**建議：**

- 等待效能補丁
- 或降低畫質設定
- PC 玩家確認硬體規格

## 戰鬥系統評價

### 近戰評價

**中規中矩：**

> 「單純的近戰本身只是及格，儘管有多種武器可選，從雙劍到大錘到刺刀。額外的功能幫助增添動作的趣味。」

**真正的亮點：**

- Formae 技能系統
- 搭檔配合
- Drain Attack 的吸血快感

### 難度設計

作為類魂遊戲，Code Vein 2 嘗試 **軟化難度**：

- 搭檔系統降低單人難度
- 開放世界讓你可以先探索其他區域再回來
- 多種 Build 選擇提供不同難度體驗

## 常見問題 FAQ

### Q1: 需要玩過前作嗎？

不需要。Code Vein 2 的劇情大部分獨立，新玩家可以直接遊玩。

### Q2: 效能問題會修嗎？

應該會。Bandai Namco 通常會在發售後推出補丁。建議關注官方公告。

### Q3: 有中文嗎？

支援繁體中文字幕。

### Q4: 可以單人遊玩嗎？

可以。搭檔系統讓單人遊玩更容易，不需要線上連線。

### Q5: PC 配備需求高嗎？

使用 UE5，建議中高階配備。目前效能問題可能讓低階配備更吃力。

### Q6: 有多人模式嗎？

支援線上合作模式。

## 重點整理

- **Code Vein 2 於 1 月 30 日發售**，帶來開放世界和時空穿越劇情
