## 最好玩的一版 DQ7

《勇者鬥惡龍 VII Reimagined》已於 2 月 5 日正式發售。這不是簡單的畫面升級，而是 Square Enix 從頭打造的全面重製，包括全新的戰鬥系統、Moonlighting 雙職業機制，以及徹底重新設計的遊戲節奏。

**Metacritic 81 分、OpenCritic 85 分**。評測一致認為：這是《DQ7》有史以來最好玩的版本，也是 2026 年初最值得投入時間的 JRPG。

如果你是 DQ 系列粉絲，這篇會幫你了解所有新變化。如果你從未玩過 DQ 系列，這篇會幫你判斷是否適合入坑，以及如何順利開局。

## 與原版的主要差異

### 視覺重製

| 項目     | PS1 原版 / 3DS 版 | Reimagined 版                 |
| -------- | ----------------- | ----------------------------- |
| 畫面風格 | 2D 像素 / 低模 3D | 高品質 3D，鳥山明風格精緻模型 |
| 環境設計 | 平面地圖          | 透視模型風格的立體場景        |
| 動畫     | 簡單攻擊動作      | 全新戰鬥動畫與演出            |
| UI 介面  | 傳統選單          | 現代化介面，資訊更清晰        |

### 遊戲節奏

原版《DQ7》最大的爭議就是**太長太慢**。100 小時以上的遊戲時間，大量重複對話和回頭路。

Reimagined 的改進：

- **刪減冗餘對話**：保留核心劇情，剪掉裝飾性文字
- **精簡任務流程**：減少不必要的跑腿
- **加快節奏**：戰鬥和過場動畫更流暢
- **預估遊戲時長**：60-80 小時（仍然很長，但比原版精簡了約 30%）

## 轉職系統完整解析

### 基礎概念

轉職（Vocation）系統是《DQ7》的核心玩法之一。玩家可以為每個角色選擇不同的職業，學習對應的技能和法術。

**重要變更：** 在 Reimagined 版本中，當你更換職業時，**不會保留**前一個職業的技能。這與原版不同，也是最大的爭議點。

### 職業結構

全遊戲共有 **26 種職業**，分為四大類：

#### 基礎職業（Basic Vocations）

入門級職業，可在 Alltrades Abbey 自由切換：

| 職業     | 英文名         | 特色             | 適合       |
| -------- | -------------- | ---------------- | ---------- |
| 戰士     | Warrior        | 高攻擊、高防禦   | 前排坦克   |
| 法師     | Mage           | 攻擊魔法         | 遠程輸出   |
| 僧侶     | Priest         | 回復魔法         | 治療輔助   |
| 武鬥家   | Martial Artist | 快速物理攻擊     | 物理輸出   |
| 盜賊     | Thief          | 速度快、偷竊技能 | 探索與刷寶 |
| 吟遊詩人 | Minstrel       | 輔助技能         | 團隊增益   |
| 牧羊人   | Shepherd       | 防禦與輔助       | 特殊情境   |
| 水手     | Sailor         | 攻防平衡         | 多功能     |

#### 中階職業（Intermediate Vocations）

需要精通兩個基礎職業才能解鎖：

| 中階職業 | 需求職業          | 特色                |
| -------- | ----------------- | ------------------- |
| 聖騎士   | 戰士 + 僧侶       | 攻防兼備，可回復    |
| 賢者     | 法師 + 僧侶       | 全能法術使用者      |
| 遊俠     | 盜賊 + 武鬥家     | 高速高攻擊          |
| 魔劍士   | 戰士 + 法師       | 物理 + 魔法混合攻擊 |
| 超級明星 | 吟遊詩人 + 武鬥家 | 強力輔助與特殊技能  |
| 海盜     | 水手 + 盜賊       | 攻擊型多功能        |
| 怪物大師 | 牧羊人 + 盜賊     | 控制與操縱怪物      |
| 機械師   | 水手 + 牧羊人     | 特殊技能與陷阱      |

#### 高階職業（Advanced Vocations）

需要精通特定的中階職業組合：

| 高階職業   | 需求                 | 特色             |
| ---------- | -------------------- | ---------------- |
| 勇者       | 特定條件（劇透注意） | 全能型，最強職業 |
| 天地魔鬥士 | 聖騎士 + 遊俠        | 極致物理戰鬥     |
| 大魔導師   | 賢者 + 魔劍士        | 極致魔法         |

## Moonlighting 系統：雙職業的革命

### 什麼是 Moonlighting？

Moonlighting 是 Reimagined 版本最重要的新機制。它允許角色**同時持有兩個職業**，學習兩邊的技能。

### 解鎖條件

- 招募到第四位永久隊員 **Aishe** 後
- 前往 **Alltrades Abbey**
- 和 **Jacqui** 對話即可解鎖

### 運作方式

| 項目           | 說明                              |
| -------------- | --------------------------------- |
| 主職業         | 決定角色的基礎數值傾向            |
| 副職業（Moon） | 提供額外技能學習                  |
| 經驗分配       | 戰鬥經驗同時計入兩個職業          |
| 切換限制       | 可隨時在 Alltrades Abbey 更換組合 |

### 推薦 Moonlighting 組合

#### 前期（剛解鎖時）

| 角色建議 | 主職業 | 副職業 | 理由                       |
| -------- | ------ | ------ | -------------------------- |
| 主角     | 戰士   | 僧侶   | 攻防兼備，同時練聖騎士前置 |
| 法師角色 | 法師   | 僧侶   | 同時練賢者前置，攻守一體   |
| 物理角色 | 武鬥家 | 盜賊   | 同時練遊俠前置，高速高輸出 |

#### 中期（解鎖中階職業後）

| 角色建議 | 主職業 | 副職業 | 理由                     |
| -------- | ------ | ------ | ------------------------ |
| 坦克     | 聖騎士 | 戰士   | 極致防禦，同時維持攻擊力 |
| 治療     | 賢者   | 僧侶   | 全能法術，回復量最大化   |
| 輸出     | 遊俠   | 武鬥家 | 極致物理輸出             |

## Let Loose 系統

### 什麼是 Let Loose？

每個職業都有一個專屬的「Let Loose」大招。當角色在戰鬥中累積足夠的「激昂值」後，就可以釋放這個強力技能。

### 如何觸發

1. **累積激昂值**：透過攻擊、受傷、使用技能來累積
2. **達到門檻**：激昂值滿後，Let Loose 選項出現
3. **釋放**：選擇使用，發動職業專屬大招

### 各職業的 Let Loose 效果

| 職業   | Let Loose 效果        | 被動效果       |
| ------ | --------------------- | -------------- |
| 戰士   | 全力斬擊，無視防禦    | 攻擊力微幅提升 |
| 法師   | 大範圍魔法爆發        | 魔法傷害增強   |
| 僧侶   | 全隊大回復 + 解除異常 | 回復效果增強   |
| 武鬥家 | 連續拳擊，多段傷害    | 暴擊率提升     |
| 盜賊   | 必定偷取稀有道具      | 掉落率提升     |

**戰術價值：** Let Loose 不只是「按下去就贏」的大招。選擇在**正確的時機**使用，是戰鬥策略的重要一環。Boss 戰中，Save Let Loose 到關鍵時刻可以扭轉戰局。

## 新手最佳開局建議

### 第一步：玩 Demo

**強烈建議先下載免費 Demo。** Demo 涵蓋遊戲開頭的 Ballymolloy 篇章，足以判斷遊戲風格是否適合你。

**Demo 特典：** 完成 Demo 後購買正式版，會解鎖「Maribel 的休閒服」造型獎勵。

### 第二步：選擇平台

| 考量     | 推薦平台              |
| -------- | --------------------- |
| 畫質優先 | PS5 / PC              |
| 攜帶遊玩 | Switch 2 / Steam Deck |
| 預算優先 | Switch（非 Switch 2） |

**特別提醒：** Switch 2 版本是這款遊戲的理想平台之一。JRPG 非常適合在通勤、睡前等零碎時間遊玩。

### 第三步：前期隊伍建議

遊戲前期角色有限，建議：

1. **主角**：保持平衡發展，不要太偏科
2. **Maribel**：偏向法術，前期主要的傷害和回復來源
3. **Ruff**：物理輸出，速度快
4. **Aishe**：加入後解鎖 Moonlighting，可依需求調整

### 第四步：難度選擇

Reimagined 版本提供多種難度修改器：

- **新手**：建議開啟「敵人弱化」和「經驗加成」
- **一般玩家**：使用預設難度
- **老手**：可開啟「強化敵人」追求挑戰

### 第五步：轉職優先順序

1. **先讓主角和 Maribel 練「戰士 + 僧侶」**：聖騎士是前中期最穩的職業
2. **讓 Ruff 練「武鬥家 + 盜賊」**：遊俠的高速高攻擊非常實用
3. **Aishe 自由發展**：她加入時可以根據隊伍需求調整

## FAQ

### Q1：沒玩過 DQ 系列，可以從這款入手嗎？

可以。每款 DQ 都是獨立故事，不需要玩過前作。但如果完全沒玩過 JRPG，建議從 DQ11（更現代、更親民）入門。

### Q2：原版的存檔可以轉移嗎？

不行。這是完全重製的新遊戲，不支援舊版存檔轉移。Demo 的進度可以轉移到正式版。

### Q3：轉職不保留技能，會不會很麻煩？

確實是最大的爭議。但 Moonlighting 系統讓你可以同時學兩個職業的技能，在一定程度上彌補了這個問題。適應後其實增加了策略深度——你需要更仔細地規劃職業路線。

### Q4：Switch 和 Switch 2 版本差異大嗎？

主要差異在畫質和幀率。Switch 2 版本畫質更高、載入更快。但 Switch 版本也能完整體驗遊戲內容，如果你不在意畫質差異，Switch 版也是好選擇。

### Q5：值得買豪華版嗎？

如果你確定會玩，2 天早期存取和裝備套組有點價值。但美術集和原聲帶精選對大多數玩家來說不是必要的。建議先玩 Demo，確定喜歡再決定版本。
