## 惡靈古堡系列的重要轉捩點

《惡靈古堡 安魂曲》（Resident Evil Requiem）是 Capcom 於 2025 年 6 月 Summer Game Fest 正式公開的系列第九款正傳作品。它不只是換個主角的續作，而是首次在正傳中採用「雙主角」設計，讓生存恐怖與動作戰鬥兩種風格並存。

遊戲將於 **2026 年 2 月 27 日**正式發售，登陸 PS5、Xbox Series X|S、Nintendo Switch 2、Steam、Epic Games Store，以及 GeForce NOW 雲端遊戲。

## 劇情設定

### 時間與地點

故事發生在浣熊市（Raccoon City）——是的，那個被核彈抹平的城市。這次的舞台是城市廢墟中一座被封鎖的 **Wrenwood Hotel**。

### 新主角：Grace Ashcroft

Grace 是 FBI 情報分析員，被派往調查 Wrenwood Hotel 內一連串離奇死亡事件。這個任務對她來說格外沉重，因為她的母親 **Alyssa Ashcroft** 八年前正是死在這裡。

Alyssa Ashcroft 是系列老玩家熟悉的名字——她是《惡靈古堡：乙女座》的可玩角色之一，浣熊市事件的倖存者，一名堅持追查 Umbrella 真相的記者。

### 老將回歸：Leon S. Kennedy

Leon 是系列最具代表性的角色之一，這次以 DSO（Division of Security Operations）特工身份回歸。這是他自《惡靈古堡 6》以來首次在正傳中擔任可玩角色。

回到浣熊市——這個他噩夢開始的地方——對這位身經百戰的特工來說意味著什麼？遊戲將深入探索他的心路歷程。

### 核心謎團：Elpis

「Elpis」是貫穿整個故事的關鍵字。它連結了：

- Alyssa Ashcroft 的死亡
- Grace 的過去
- Leon 的調查任務

玩家需要在雙主角的交織視角中，逐步揭開這個謎團的真相。

## 雙主角系統：兩種截然不同的遊戲體驗

這是《安魂曲》最大的特色。遊戲會在 Grace 和 Leon 之間切換，類似《惡靈古堡：乙女座》的設計，但兩人的玩法完全不同。

### Grace Ashcroft：生存恐怖

Grace 的部分是純正的「生存恐怖」體驗：

**核心玩法：**

- 資源極度稀缺，需要謹慎管理
- 強調潛行與躲避，而非正面戰鬥
- 有一隻怪物會全程追獵玩家（類似暴君 Mr. X 或 Lady Dimitrescu）

**獨特機制：**

| 機制     | 說明                                               |
| -------- | -------------------------------------------------- |
| 潛行系統 | 蹲下移動減少噪音，可躲藏在桌子等物件下方           |
| 打火機   | 照亮黑暗區域，但會吸引怪物注意                     |
| 玻璃瓶   | 收集並投擲來分散怪物注意力                         |
| 血液工藝 | 使用感染敵人的血液製作彈藥、道具和一擊必殺的注射器 |

**恐怖氛圍：**
Grace 的章節預設使用第一人稱視角，營造更強的沉浸感和壓迫感。資源有限、怪物追獵、黑暗環境——這是系列回歸恐怖本源的設計。

### Leon S. Kennedy：動作戰鬥

Leon 的部分延續《惡靈古堡 4》以來的動作風格：

**核心玩法：**

- 資源相對充足
- 強調流暢的戰鬥與連技
- 槍械搭配近戰，節奏明快

**獨特機制：**

| 機制       | 說明                                                 |
| ---------- | ---------------------------------------------------- |
| 手斧       | 新增的近戰武器，可格擋、造成大量傷害、撬開上鎖的門櫃 |
| 銳利度系統 | 手斧有耐久度，需要維護以保持最佳傷害                 |
| 奪取武器   | 可從敵人手中奪取武器並反擊                           |
| 經典槍械   | 手槍、霰彈槍等系列招牌武器                           |

**動作導向：**
Leon 的章節預設使用第三人稱視角，更適合展現戰鬥動作。這是給喜歡《惡靈古堡 4》風格的玩家準備的。

## 視角切換系統

《安魂曲》的一大創新：玩家可以**隨時切換第一人稱和第三人稱視角**。

| 視角     | 預設角色 | 特點                   |
| -------- | -------- | ---------------------- |
| 第一人稱 | Grace    | 沉浸感強、恐怖氛圍濃厚 |
| 第三人稱 | Leon     | 適合動作戰鬥、視野較廣 |

Capcom 表示：

- 第一人稱提供「緊張、寫實的遊戲體驗」
- 第三人稱「適合喜歡動作導向的玩家」

你可以用第一人稱玩 Leon，也可以用第三人稱玩 Grace——完全取決於個人喜好。

## 難度設定

### 休閒模式（Casual）

- 開啟輔助瞄準
- 玩家血量更高
- 敵人耐久度降低

### 標準模式（Standard / Classic）

- 傳統生存恐怖體驗
- Grace 章節需要使用**墨水緞帶**才能存檔（經典設計回歸！）
- 需要策略性管理資源和存檔時機

## 值得期待的看點

### 1. 回到浣熊市

這是系列首次讓玩家「回到」浣熊市（在它被摧毀之後）。Wrenwood Hotel 內會有多少對舊作的致敬和彩蛋？

### 2. Leon 的角色深度

Leon 從《惡靈古堡 2》的新人警察，到現在的資深特工。這次回到噩夢開始的地方，遊戲將探索他這些年的心理變化。

### 3. Alyssa 的故事延續

《乙女座》的 Alyssa Ashcroft 終於有了後續。她的女兒 Grace 會揭開什麼真相？

### 4. 雙風格設計

喜歡恐怖的玩 Grace，喜歡動作的玩 Leon。Capcom 試圖同時滿足兩種玩家群體。

## FAQ

### Q1：需要玩過前作嗎？

不需要，但玩過會更能理解彩蛋和角色背景。特別是《惡靈古堡 2》（Leon 初登場）和《乙女座》系列（Alyssa 的故事）。

### Q2：Grace 和 Leon 的章節可以選擇順序嗎？

不行。遊戲會自動在兩人之間切換，類似《乙女座》的設計。

### Q3：有中文嗎？

有。支援繁體中文和簡體中文字幕，語音有英文和日文可選。

### Q4：Switch 2 版表現如何？

目前未有詳細規格，但同步發售顯示 Capcom 對移植品質有信心。建議等發售後的實機評測。

### Q5：有多人模式嗎？

目前公開的資訊顯示這是單人遊戲。額外模式尚未公布。
