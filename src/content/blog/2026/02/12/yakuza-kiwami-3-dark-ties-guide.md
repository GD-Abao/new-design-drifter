## 一套價格，兩款遊戲

《人中之龍 極 3 & Dark Ties》（Yakuza Kiwami 3 & Dark Ties）今天正式發售。這不只是《人中之龍 3》的簡單重製——SEGA 的龍が如くスタジオ（Ryu Ga Gotoku Studio）打包了兩款完整遊戲：

- **Kiwami 3：** 經典《人中之龍 3》的現代化重製，以桐生一馬為主角
- **Dark Ties：** 全新前傳遊戲，以原作反派峯義孝（Mine）為主角

評測成績褒貶不一，有些媒體讚揚其「誠意十足的重製」，也有聲音認為「系列需要休息一下」。本文幫你釐清這款遊戲到底值不值得買。

## Kiwami 3：桐生一馬在沖繩的故事

### 劇情概要

故事承接《人中之龍 2》結尾。桐生一馬離開東城會，帶著遙在沖繩開設了兒童養育院「牽牛花」（あさがお），過著平靜的生活。

然而，政治家與黑道的陰謀再次將他捲入紛爭。一連串事件牽扯到東城會的存亡、CIA 的介入，以及神秘的商人峯義孝。

### 與原版的差異

| 項目       | 原版（PS3）      | Kiwami 3（2026）            |
| ---------- | ---------------- | --------------------------- |
| 引擎       | 舊引擎           | Dragon Engine 改良版        |
| 畫面       | 720p 時代        | 4K 支援、光影效果大幅提升   |
| 戰鬥系統   | 單一風格         | 雙風格（堂島之龍 + 龍宮流） |
| 被刪減內容 | 曾被批評大量刪減 | 恢復部分被刪減的支線任務    |
| 追逐戰     | 原版的痛點       | 重新設計，操控更順暢        |
| 格擋系統   | 原版的痛點       | 重新平衡，不再過於強勢      |

### 全新戰鬥風格：龍宮流

Kiwami 3 最大的新增內容是**龍宮流**（Ryukyu Style）——一套全新的武器型戰鬥風格。

**堂島之龍 vs 龍宮流：**

| 風格     | 堂島之龍（Dragon of Dojima） | 龍宮流（Ryukyu Style） |
| -------- | ---------------------------- | ---------------------- |
| 武器     | 拳頭為主                     | 劍與盾                 |
| 防禦     | 閃避為主                     | 盾牌格擋               |
| 攻擊範圍 | 近身                         | 中距離                 |
| 特殊技   | 經典 Heat Action             | 範圍攻擊、破防技       |
| 適合場景 | 一對一 Boss 戰               | 多人混戰               |

**龍宮流的獨特技能：**

- **範圍攻擊（Area-of-Effect）：** 劍的橫掃可以同時打擊多個敵人
- **破防技（Guard Break）：** 破除持盾或格擋的敵人
- **盾擊（Shield Bash）：** 用盾牌撞擊敵人造成暈眩
- **切換連技：** 從龍宮流切換到堂島之龍時有特殊連擊

### 飆車女王模式（Bad Boy Dragon）

Kiwami 3 的專屬大型支線模式。

**內容：**

- 桐生一馬率領一群女騎士團隊
- 參與大規模多隊伍飆車戰鬥
- 招募和自訂騎士成員
- 改裝機車

**評測反應：**

> 「自訂和招募騎士非常有趣，大規模戰鬥的場面也很壯觀。」— RPGFan

這是一個意外的亮點，為主線劇情之外提供了大量的可玩內容。

## Dark Ties：峯義孝的前傳

### 劇情概要

Dark Ties 是一個全新的前傳故事，主角是《人中之龍 3》的反派——**峯義孝（Yoshitaka Mine）**。

故事追蹤 Mine 如何從一個無名小卒，一步步攀升到東城會的權力核心。玩家將從他的視角理解他的動機、掙扎和最終走上反派之路的過程。

### 為什麼 Mine 值得一個前傳？

Mine 是《人中之龍 3》中最具爭議的反派之一。許多玩家認為他的動機在原作中交代不足，讓最終 Boss 戰缺乏情感衝擊。

Dark Ties 試圖彌補這個缺憾：

> 「Dark Ties 在展示 Mine 的神秘心理方面做得非常出色，補完了原作中最大的遺憾。」— TheSixthAxis

### Dark Ties 獨有系統：地下格鬥場

Dark Ties 的專屬大型支線模式。

**核心機制：**

| 項目 | 說明                     |
| ---- | ------------------------ |
| 類型 | Roguelike 地牢探索       |
| 結構 | 隨機生成的多層迷宮       |
| 戰鬥 | 尋找武器和升級道具       |
| 同伴 | 招募傭兵同伴協助戰鬥     |
| 進度 | 成功逃脫後可永久升級角色 |

**與主線的差異：**

主線是線性劇情驅動，地下格鬥場是可反覆遊玩的隨機內容。如果你喜歡 Roguelike 元素，這個模式可以提供數十小時的額外遊戲時間。

## 需要玩過前作嗎？

### 短回答

**Kiwami 3：** 建議玩過《人中之龍 0》→《極 1》→《極 2》，但不是必要條件。遊戲有劇情回顧功能。

**Dark Ties：** 建議至少玩過 Kiwami 3 的主線，因為 Dark Ties 是其前傳。

### 系列入門建議

| 你的情況                | 建議                             |
| ----------------------- | -------------------------------- |
| 從未玩過人中之龍        | 先從《人中之龍 0》入門           |
| 玩過 0 和極 1、極 2     | 直接玩 Kiwami 3，再玩 Dark Ties  |
| 只玩過《如龍 7》或《8》 | 可以玩，但桐生的故事線可能不完整 |
| 時間有限                | 可以只玩 Dark Ties（15-20 小時） |

## 值得買嗎？

### 推薦購買

- **人中之龍系列粉絲：** 龍宮流戰鬥風格和 Dark Ties 前傳故事是值回票價的新內容
- **想補完系列的玩家：** Kiwami 3 是系列劇情中重要的一環
- **喜歡日式犯罪劇情的玩家：** 沖繩的故事氛圍獨特

### 考慮觀望

- **系列疲勞的玩家：** 如果你覺得《如龍》系列重製太頻繁，這一作可能不會改變你的想法
- **預算有限：** 2 月有很多大作（Nioh 3、惡靈古堡安魂曲），這款的優先度可以排後面
- **只對新作有興趣：** 畢竟是 2009 年遊戲的重製，核心劇情不是全新的

### 不推薦

- **從未接觸過系列：** 直接從 Kiwami 3 開始會錯過大量劇情脈絡
- **不喜歡日式動作冒險：** 這款遊戲的核心體驗沒有根本性改變

3. **競爭激烈：** 2 月的遊戲陣容太強，相對之下這款不那麼突出

## FAQ

### Q1：一定要兩款都玩嗎？

不一定。你可以先玩 Kiwami 3 主線，再決定是否玩 Dark Ties。但建議至少玩一下 Dark Ties 的主線——它對 Mine 的塑造非常出色，會讓你重新評價原作的結局。

### Q2：Switch 2 版本值得買嗎？

如果你只有 Switch 2，可以買。畫質有妥協但可接受，30fps 對劇情導向的遊戲影響不大。如果你同時擁有 PS5 或 PC，建議選那邊。

### Q3：地下格鬥場難嗎？

中等難度。Roguelike 元素意味著會有隨機性，但成功逃脫後的永久升級會讓每次嘗試都變得更容易。不是硬核 Roguelike，更像是一個輕鬆的加分模式。

### Q4：有多人模式嗎？

沒有。這是純單人遊戲。

### Q5：會有後續的 Kiwami 4 嗎？

目前沒有官方消息，但考慮到 SEGA 已經重製了 1-3，Kiwami 4 應該只是時間問題。《人中之龍 4》有四位主角的獨特結構，重製版會很有趣。

## 重點整理

1. **雙遊戲套裝 \$49.99**：Kiwami 3（桐生重製，25-30 小時）+ Dark Ties（Mine 前傳，15-20 小時），總計約 40-50 小時內容
2. **龍宮流是最大亮點**：劍與盾的全新戰鬥風格，讓多人混戰更有策略性，與堂島之龍的切換連技增加了深度
3. **Dark Ties 補完 Mine 的故事**：評測一致認為對反派的心理描寫出色，Roguelike 地下格鬥場提供額外可玩性
4. **評價中上但有爭議**：新內容獲得肯定，但部分媒體對系列重製疲勞表示擔憂
5. **建議先玩前作**：至少玩過《人中之龍 0》和《極 1、2》再入手，體驗會好很多
6. **平台推薦 PS5 或 PC**：60fps 體驗最佳，Switch 2 適合重視攜帶性的玩家
